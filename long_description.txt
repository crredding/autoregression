galgraphs
=====

Provides
  1. Graphing tools to plot arrays and pandas dataframes
  2. Run linear regressions of natural splines, and visualize the results next to the data
  3. Visualize the errors

How to use the documentation
----------------------------
Documentation in docstrings provided with the code.

We recommend exploring the docstrings using
`IPython <http://ipython.scipy.org>`_, an advanced Python shell with
TAB-completion and introspection capabilities.  See below for further
instructions.

The docstring examples assume that `galgraphs` has been imported as `gal`::

  >>> import galgraphs as gal

Suggested install at

$ pip install git+https://github.com/pgeurin/galgraphs

Code snippets are indicated by three greater-than signs::

  >>> x = 42
  >>> x = x + 1

Use the built-in ``help`` function to view a function's docstring::

  >>> help(np.sort)
  ... # doctest: +SKIP
